# 🔐 Password Generator

A modern and customizable **Password Generator** built using **HTML, CSS, and JavaScript**.  
This tool helps users create strong, secure passwords instantly with multiple customization options.

---

## 🚀 Features

- Generate **strong random passwords**
- Adjustable **password length**
- Option to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- **Password strength indicator**
- **Copy to clipboard** functionality
- Clean and responsive UI

## ⚙️ How It Works

1. Choose your **password length**.
2. Select the **character types** you want:
   - Uppercase
   - Lowercase
   - Numbers
   - Symbols
3. Click **Generate Password**.
4. Copy the generated password using the **Copy** button.
